# Fonts

This folder bundles **Inter**, licensed under the SIL Open Font License (OFL). The font files were sourced from Google Fonts:

- https://github.com/google/fonts/raw/main/ofl/inter/Inter%5Bopsz%2Cwght%5D.ttf
- https://raw.githubusercontent.com/google/fonts/main/ofl/inter/OFL.txt

`Inter-Regular.ttf`, `Inter-Bold.ttf`, and `Inter-Black.ttf` are saved with the exact filenames expected by the engine. They use the Inter variable TTF, which includes regular through black weights.

You can override these defaults in `config.json` with `config.font` and `config.font_bold` paths.
